# Drawing Lines

A line is drawn using two points. We can also set the outline colour and width of the line. The following code demonstrates. **Remember to create a graphics window first**.

```python
line = Line(Point(0, 0), Point(600, 400))
line.setOutline('red')
line.setWidth(80)
line.draw(win)
```
